# pause-privacy
Pause: Christian Porn Recovery (CPR) Privacy Policy

Effective date: June 15, 2026

Pause: Christian Porn Recovery (CPR) is a Chrome extension that creates a private pause when pornographic or adult content is detected. Its tagline is: "Recovery starts with a pause."

## No Data Collection

Pause CPR does not collect, sell, rent, transmit, store, or share personal data.

The extension does not use:

- Accounts
- Subscriptions
- Analytics
- Advertising
- Tracking pixels
- Remote logging
- External databases
- Background data uploads
- Cloud services
- Telemetry

## Local Processing

Pause CPR checks page information locally in your browser to decide whether to show a Scripture reminder. This processing happens on your device.

The extension may read the current page URL, page title, page text, search query text, headings, metadata, and simple media/embed signals. This information is used only in the moment, inside your browser, to decide whether the reminder should appear.

Pause CPR does not send this information anywhere.

## Local Storage

Pause CPR uses Chrome's local extension storage only to remember whether the reminder was temporarily dismissed. This setting stays on your device and is not transmitted to the developer or any third party.

## Third-Party Links

The "Read full chapter/The Bible" button opens ESV.org in a new tab. ESV.org is operated by Crossway and has its own privacy practices. Pause CPR does not share browsing data with ESV.org; opening the link is a normal user-initiated website visit.

## Remote Code

Pause CPR does not load or execute remote code.

## Contact

For privacy questions, contact the developer through the Chrome Web Store listing.
